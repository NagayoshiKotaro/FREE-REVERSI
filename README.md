# FREE-REVERSI

## To Start GAME 
please install pygame and mutagen for example
```html:sample
    $ pip install pygame
    $ pip install mutagen
```
to play game
```html:sample
    $ python reversi.py
```
if this don't work please try
```html:sample
   $ pip install -r requirments.txt
```
